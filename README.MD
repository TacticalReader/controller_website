
## GameSphere GS4 Controller - Landing Page

**GameSphere GS4 Controller Landing Page** is an interactive, feature-rich website showcasing a next-generation gaming controller with advanced haptic feedback and adaptive triggers. The project demonstrates modern web development practices with dynamic UI elements, smooth animations, and accessibility features.

### Features

The landing page includes several interactive sections designed to engage visitors and showcase the controller's capabilities:

- **Dynamic Hero Section** - Eye-catching introduction with call-to-action buttons and smooth scroll navigation
- **Interactive Feature Cards** - Hover-reveal cards highlighting 4K/120fps gameplay, ultra-fast SSD, 3D audio, dynamic haptics, adaptive triggers, and ergonomic design with heatmap visualization
- **Color Selector** - Military-style loadout interface allowing users to switch between Midnight Black, Glacier White, and Magma Red controller variants with sound effects
- **Technical Specifications** - Expandable spec cards with detailed information about haptic feedback, adaptive triggers, connectivity, battery life, weight, and built-in microphone
- **Testimonials Section** - Dynamic testimonial wall featuring user reviews with ratings, verification badges, and play hours loaded from JSON data
- **Newsletter Subscription** - Functional email capture form with validation feedback
- **Login Modal** - Animated login interface with social media integration options
- **Mission Toast** - Gamified countdown timer encouraging user engagement
- **Developer Credits** - Modal showcasing project contributors
- **Scroll Progress Bar** - Visual indicator of page scroll position
- **Smooth Scrolling** - Seamless navigation between page sections
- **Responsive Design** - Mobile-friendly hamburger menu and adaptive layouts
- **Accessibility Features** - ARIA labels, keyboard navigation support, skip links, and screen reader compatibility

### Technologies Used

- **HTML5** - Semantic markup with accessibility attributes
- **CSS3** - Modern styling with custom properties, animations, gradients, and flexbox/grid layouts
- **JavaScript (Vanilla)** - Dynamic content loading, event handling, and interactive features
- **Font Awesome 6.2.0** - Icon library for enhanced visual elements
- **Google Fonts** - Mulish and Roboto Mono font families
- **JSON** - Data-driven testimonials

### Getting Started

#### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies or build tools required

#### Installation

1. Clone the repository:
```bash
git clone https://github.com/TacticalReader/controller_website.git
```

2. Navigate to the project directory:
```bash
cd controller_website
```

3. Open `index.html` in your web browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Alternatively, use a local development server for the best experience:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

### Key Sections

#### Home Section
Introduces the GameSphere GS4 controller with a compelling hero image and call-to-action buttons directing users to explore features or make a purchase.

#### Features Section
Six interactive cards showcasing core capabilities including 4K/120fps gaming, ultra-fast SSD technology, 3D audio, dynamic haptics, adaptive triggers, and an ergonomic heatmap visualization.

#### Color Selector
Military-themed loadout interface with tactical reticles allowing users to preview three controller color variants with accompanying sound effects.

#### Technical Specifications
Detailed specs presented in expandable cards covering haptic feedback technology, adaptive trigger mechanics, Bluetooth 4.0 connectivity, 12-hour battery life, 280g weight, and integrated microphone.

#### Testimonials
User reviews from gamers, streamers, and tech reviewers displaying ratings, verification status, hours played, and detailed feedback loaded dynamically from JSON.
